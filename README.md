# androidAppTemplate
Simple template to create android apps - includes mainly used libraries in the gradle file

## Library list: 
- [AndroidAnnotations](https://github.com/excilys/androidannotations/)
- [Logger](https://github.com/orhanobut/logger)
- [EventBus](https://github.com/greenrobot/EventBus)
- [GSON](https://code.google.com/p/google-gson/)
- [ACRA](https://github.com/ACRA/acra)
- [Retrofit](https://github.com/square/retrofit)
- [Picasso](https://github.com/square/picasso)
- [AndroidSaripaar](https://github.com/ragunathjawahar/android-saripaar)
- [ApacheCommons](https://commons.apache.org/)
- [Advanced Recycler View](https://github.com/h6ah4i/android-advancedrecyclerview)
- [Guava](https://github.com/google/guava)
- [Once](https://github.com/jonfinerty/Once)
- [RxJava](https://github.com/ReactiveX/RxJava)
- [RxAndroid](https://github.com/ReactiveX/RxAndroid)
- [JodaTimeAndroid](https://github.com/dlew/joda-time-android)
